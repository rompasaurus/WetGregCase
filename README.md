# WetGregCase

A parametric, two-piece **snap-fit enclosure** for the
[Dilder](https://github.com/rompasaurus) **Full PCB Rev 1**, generated by a
FreeCAD macro.

- **Front cover** (screen side) — deep, with an **aggressively curved front edge**
  so it reads as thin; carries the screen window and the joystick hole/thumbpiece.
  Its outer face **meets the top of the screen**.
- **Backplate** (Pico side) — **thin**, only as tall as the USB-C connector, a
  perimeter frame with a small **shelf the PCB rests on** and an open centre so it
  never touches a component.
- **Thumbpiece** — snap cap over the 5-way joystick (rev2-mk2 style).

The macro **imports the real KiCad STEP assembly**, aligned to the case — with the
unpopulated header pins **stripped** and the screen **shifted to meet the front
face** — so the fit is checked against actual geometry.

## Key facts

| | |
|---|---|
| Board | 97.8 × 42.0 × 1.6 mm, R5 corners |
| Screen face | meets screen top at Z = −11.4 |
| Backplate height | = USB-C connector (Z = +4.9) |
| Outer footprint | 102.4 × 46.6 mm (both plates, flush) |
| Overall height | **16.3 mm** |
| Front curve | r = 6 mm |
| Closure | flush lap joint + 6 snap detents + PCB shelf |
| Controls | side USB-C opening · −Y power-switch slot · joystick thumbpiece |

## Layout

```
WetGregCase/
├─ freecad/
│  ├─ wetgreg_case.FCMacro   # parametric generator (edit PARAMS, re-run)
│  └─ WetGregCase.FCStd      # generated assembly
├─ exports/
│  ├─ WetGregCase_FrontCover.step
│  ├─ WetGregCase_BackPlate.step
│  └─ WetGregCase_Thumbpiece.step
├─ MODEL-SPEC.md             # full design spec, dimensions, open items
└─ README.md
```

## Run it

```bash
cd freecad && freecadcmd wetgreg_case.FCMacro   # headless regenerate
# or open the macro in the FreeCAD GUI and Execute
```

See **[MODEL-SPEC.md](MODEL-SPEC.md)** for full dimensions, the parameter
reference, and open items to confirm during validation.
